# MarkItDown Mirror Setup

This repository is prepared to mirror the upstream Microsoft MarkItDown repository:

- Upstream: https://github.com/microsoft/markitdown
- Purpose: convert documents such as PDF, Word, Excel, PowerPoint, images, audio, HTML, CSV, JSON and XML into Markdown for AI review, Notion import, and technical knowledge management workflows.

## Recommended Use

This repository is intended for a controlled mirror/copy of the upstream project, while keeping attribution to Microsoft and preserving the upstream license after synchronization.

## How to Sync Upstream

1. Go to **Actions**.
2. Select **Sync upstream markitdown**.
3. Click **Run workflow**.
4. After completion, this repository will be replaced with the latest content from `microsoft/markitdown` main branch.

## Note

This is not a native GitHub fork created through the GitHub fork network. It is a mirror-sync setup because the available GitHub connector does not expose a fork operation.
